# Viewport KB

Public-safe, Obsidian-compatible knowledge base for Viewport intake.

Rules:
- Never commit raw Telegram/session content.
- Store paraphrased captures only.
- No secrets, tokens, credentials, cookies, or private keys.
- Raw session DB remains sensitive and local-only.

Folders:
- `ideas/` raw ideas and product concepts
- `references/` fetched/summarized links/images/videos
- `decisions/` decision and architecture logs
- `tasks-from-chat/` paraphrased task captures
